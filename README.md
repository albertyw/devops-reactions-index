Index of DevOps Reactions

## Installation

1.  Install Go 1.6.
2.  Install [Glide](https://github.com/Masterminds/glide/blob/master/README.md)
3.  `go get -u github.com/golang/lint/golint`
4.  `go build`
5.  `./devops_reactions_index`

## Running tests
